# **big**

Build with `cargo build --release`

Usage: `target/debug/big TEXT [options]`

Options:
```
    -f, --font PATH     set font
    -s, --speed INT     set text speed (default: 10)
    -e, --effects EFFECT1 EFFECT2 ...
                        pick only some effects
    -l, --loop          loop text
    -h, --help          print help menu
```

Background effects :
 - empty

Text effects :
 - normal
 - rainbow
 - split
